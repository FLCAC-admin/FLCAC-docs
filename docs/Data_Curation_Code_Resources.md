# Data Curation Code

Below is a list of data development resources used to generate datasets through code on the Federal LCA Commons.

| Resource                                                                              | Description                                                                                        | Repository                |
|---------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------|---------------------------|
| [uslci-moves](https://github.com/FLCAC-admin/uslci-moves)                             | Converts EPA MOVES data into unit processes for on-road trucking.                                  | USLCI                     |
| [uslci-marine](https://github.com/FLCAC-admin/uslci-marine)                           | Module for generating marine transport unit processes                                              | USLCI                     |
| [uslci-transport](https://github.com/FLCAC-admin/uslci-transport)                     | Uses Census Commodity Flow Survey to estimate distances and transport modes for common commodities | USLCI                     |
| [uslci-rail](https://github.com/FLCAC-admin/uslci-rail)                               | Module for generating rail transport unit processes                                                | USLCI                     |
| [international-electricity](https://github.com/FLCAC-admin/international-electricity) | International grid mix data from Ember                                                             | _coming soon_             |
| [ElectricityLCI](https://github.com/NETL-RIC/ElectricityLCI)                          | U.S. Electricity Baseline                                                                          | U.S. Electricity Baseline |
